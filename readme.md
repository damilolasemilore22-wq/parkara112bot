# parkara112bot

A Telegram bot deployed via Railway.

## Setup
1. Clone this repo
2. Run `npm install`
3. Create `.env` from `.env.example` and add your BOT_TOKEN
4. Run `npm start`

## Deployment (Railway)
1. Push this repo to GitHub
2. Create a new Railway project → Deploy from GitHub repo
3. Add `BOT_TOKEN` in Railway's Variables tab
4. Railway will auto-deploy on push
